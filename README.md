# Automation-Selenium-Practice
Selenium Hybrid Framework built with Java, TestNG, Maven, and Log4j. Features include Page Object Model (POM), Data-Driven testing, and Jenkins CI integration.
## 🛠️ Tech Stack
* **Language:** Java (JDK 11)
* **Web Automation:** Selenium WebDriver
* **Test Runner:** TestNG
* **Build Tool:** Maven
* **Logging:** Log4j 2 (Console & File appenders)
* **CI/CD:** Jenkins
